# Backend Developer — Python / Django

**Location:** Bangalore, Remote

**Experience Required:** 2-4 years

## About the role

We are looking for a Backend Developer to help us build and scale our SaaS
platform. You'll partner closely with product and design to ship features
that thousands of customers rely on every day.

## What you'll do

- Build and maintain Django services that power our customer-facing APIs
- Design REST APIs and document them for our partner integrations
- Optimize PostgreSQL queries and database schemas for read-heavy workloads
- Collaborate with the frontend team on data contracts and pagination
- Mentor junior engineers and review code daily
- Operate services in production: monitoring, on-call, and incident response

## Things you absolutely need

- Strong Python production experience
- Django (including DRF) and REST API design
- PostgreSQL — query tuning, indexing, migrations
- Git workflows and CI/CD pipelines
- Solid debugging skills

## Things we'd like you to have

- Experience with Celery and Redis for background work
- Docker and Kubernetes for service deployment
- Familiarity with AWS or GCP
- Exposure to GraphQL or gRPC

## Qualifications

- Bachelor's degree in Computer Science or related field
- 2-4 years of professional backend engineering experience
- Strong written and verbal communication

## Things we explicitly do NOT want

- Frontend-only profiles with no backend production exposure
- Candidates with no Python production deployment history
